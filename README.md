# College Admission System (Java)

A console-based Java mini project that simulates a real-time college admission process.
The program collects student details, verifies eligibility based on marks and stream, suggests departments, and displays fee structure.

---

## Features

* Collects student details (Name, Age, Father's Name, Marks)
* Allows stream selection:

  * Bio Maths
  * Computer Science
  * Commerce
* Allows department selection:

  * Computer Science
  * Computer Application
  * Mathematics
  * Commerce
* Checks eligibility based on rules
* Displays admission result
* Shows department fee structure
* Displays an admission summary
